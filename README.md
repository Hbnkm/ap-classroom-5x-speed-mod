# AP Classroom 5x Speed Mod 🚀

A Chrome extension that enables **5x video playback speed** on AP Classroom.

## Installation

1. Clone: `git clone https://github.com/HbnkmYo/ap-classroom-5x-speed-mod`
2. Open `chrome://extensions/`
3. Enable **Developer mode**
4. Click **Load unpacked** → select folder
5. Done! 🚀

## Usage

- Press `5` = 5x speed
- Press `1-9` for other speeds
- Or click extension icon for buttons
